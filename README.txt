## Description
A C# console application designed to help learn about online safety and cyber

## Features
* **Voice Greeting:** Plays a .wav audio file on startup.
* **ASCII Art:** Displays a custom title screen logo.
* **Bot Interaction:** Asks for user's name and answers safety questions.
* **Error Handling:** Handles empty inputs and invalid queries gracefully.

## How to Run
1. Open the solution in Visual Studio.
2. Press **F5** to build and run.
